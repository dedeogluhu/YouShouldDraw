# Getting Started

### Requirements
```
Nodejs and npm or yarn package manager
```

### Project setup
```
yarn install
```

### Builds the Vue frontend
```
yarn build
```

### Runs the Expressjs server after you build the frontend
```
yarn serve
```

### Runs the Vue dev server if you don't want to work with the backend service
```
yarn client
```
